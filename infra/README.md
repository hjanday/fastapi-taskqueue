# Infrastructure

This directory contains infrastructure-as-code configurations for deploying the FastAPI Task Queue system.

## Kubernetes

Kubernetes manifests will be added in the `k8s/` directory for deploying:
- API server deployment
- Worker deployment
- Services
- ConfigMaps
- Secrets

## Terraform (Optional)

Terraform configurations for AWS resources will be added as needed for:
- SQS queues
- DynamoDB tables
- IAM roles and policies
- EKS cluster configuration

## Deployment

Deployment instructions will be added as configurations are created.
